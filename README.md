# charity_managment

Feature-first Flutter architecture for a charity management mobile app.

## Included

- Riverpod state management setup
- GoRouter navigation with role-aware redirects
- Dio networking abstraction layer
- Mock repositories and mock data sources
- Core/shared/services/theme/routing modular structure
- Donor and Charity Organization role flow scaffolding

## Docs

- Architecture details: [`ARCHITECTURE.md`](ARCHITECTURE.md)

## Run

```bash
flutter pub get
flutter run
```
